This repository contains hands-on implementations of **Classification Algorithms** in Machine Learning using Python and Jupyter Notebooks.

It focuses on **Logistic Regression, Hyperparameter Tuning, and Multiclass Classification**, helping build strong fundamentals for real-world ML problems.


## 🧠 Key Concepts Covered

- Supervised Learning  
- Classification Algorithms  
- Logistic Regression  
- Sigmoid Function  
- Decision Boundary  
- Hyperparameter Tuning  
- Model Optimization
